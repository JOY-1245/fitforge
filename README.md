A stunning, portfolio-worthy fitness and gym website built with vanilla HTML, CSS, and JavaScript. Features a dark premium theme, glassmorphism UI, interactive tools, and smooth animations.

## ✨ Features

- 🔥 **Animated Hero** — Full-screen hero with parallax background, typing effect, and floating stats
- 🏢 **About Section** — Feature cards with scroll-triggered animations
- 🏋️ **6 Programs** — HIIT, Yoga, CrossFit, Strength, Cardio, Boxing with hover effects
- 👨‍🏫 **Trainer Profiles** — 4 trainer cards with social links and ratings
- 📅 **Weekly Schedule** — Interactive tab-based class timetable with 7-day coverage
- 🧮 **BMI Calculator** — Real-time calculation with animated gauge and health tips
- 💪 **Workout Plan Builder** — 3-step wizard that generates personalized weekly plans
- 💰 **Pricing Plans** — 3-tier cards with monthly/yearly toggle and 20% savings
- ⭐ **Testimonials** — Auto-playing carousel with swipe support
- 📩 **Contact Form** — Floating labels, validation, and success animation
- 🦶 **Premium Footer** — Newsletter signup, quick links, and social media

## 🎨 Design Highlights

| Feature | Technology |
|---|---|
| Dark Theme | CSS Custom Properties |
| Glassmorphism | `backdrop-filter` + transparency |
| Scroll Animations | Intersection Observer API |
| Responsive Design | CSS Grid + Flexbox + Media Queries |
| Typography | Google Fonts (Outfit + Inter) |
| Micro-Animations | CSS transitions + keyframes |
| Custom Scrollbar | WebKit + Firefox scrollbar styling |
| Accessibility | `prefers-reduced-motion`, focus-visible, ARIA labels |

## 🚀 Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/fitforge.git
   ```

2. Open `index.html` in your browser — no build tools needed!
